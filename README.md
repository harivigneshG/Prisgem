## Prisgem Tweaker

A tiny app customize the Frappe's Admin UI / desk 


**Features**:
* Modify App Logo
* Hide Help / Notification Menu
* Make Full width by default
* Change Login Page Title Name
* Avoid Onboarding 
* Customize navbar color
